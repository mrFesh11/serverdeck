# ServerDeck

Desktop-панель управления серверами по SSH: Tauri 2 + React/TypeScript + Rust (ssh2). Ключи не покидают машину, на серверы ничего ставить не нужно.

## Возможности

- **SSH-терминал** (xterm.js, PTY), вкладки, Ctrl+1..9, живые сессии в фоне
- **Обзор сервера**: CPU / RAM / диск / сеть / load / uptime со спарклайнами и ховер-тултипами (значение + время), таблица Docker-контейнеров с просмотром логов
- **Проводник**: дерево директорий, сортировка, table/grid, скрытые файлы, превью, chmod / rename / rm / mkdir, мультивыделение, drag&drop-загрузка, поиск (мгновенный фильтр + глубокий `find` по Enter)
- **Файлы**: двухпанельный SFTP (локально ↔ сервер) с фильтрами
- **Сниппеты**: библиотека команд с тегами и поиском, запуск на нескольких серверах сразу, вывод и exit-код по каждому
- **Jump-хосты** (ProxyJump), тест соединения, палитра Ctrl+K, трей, RU/EN

## Хранение

Конфиг: `~/.config/com.serverdeck.app/config.json` — только адреса серверов и пути к SSH-ключам. Пароли и ключи не сохраняются; авторизация — ключом или через ssh-agent.

## Сборка

Зависимости: Rust (stable), Node.js 20+, на Linux — `libwebkit2gtk-4.1-dev libssl-dev libayatana-appindicator3-dev librsvg2-dev`.

```bash
npm install
npm run tauri build -- --bundles deb   # собрать установочный deb-пакет
npm run tauri dev                      # запуск из исходников с хот-релоадом (для разработки)
```

## Мониторинг

Метрики собираются по SSH раз в 7 секунд: CPU — по дельте `/proc/stat`, память/диск/сеть — `free`, `df`, `/sys/class/net`; `docker ps` — только при открытой вкладке «Обзор». При скрытом окне опрос останавливается.
